# AI Usage

AI tools were used during this assignment to support design exploration, implementation speed, review, prompt preparation, commit-message drafting, and translation.

## Tools used

- **Claude Design** — used for UI design exploration, design-system generation, screen layout ideas, and visual assets.
- **Claude Code** — used as a coding agent to speed up implementation, suggest refactorings, assist with code review, and help validate changes against the assignment requirements.
- **Gemini** — used for commit-message generation and translation assistance.
- **ChatGPT** — used for prompt preparation, prompt optimization, requirement breakdown, architectural discussion, and review planning.

## Final responsibility

All final implementation decisions, architecture choices, code structure, requirement coverage, and submitted changes were manually reviewed and adjusted by the author.
